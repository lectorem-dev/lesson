# Урок 3. Тестирование и сертификат

Тест нужен для проверки усвоения материала. В MVP используется простой формат: один вопрос и несколько вариантов ответа, среди которых только один правильный.

## Как работает тест

После чтения урока пользователь отвечает на вопросы.

Основные правила:

- в каждом вопросе один правильный ответ;
- проходной балл задается отдельно для каждого урока;
- варианты ответов перемешиваются;
- если балл ниже проходного, тест можно пройти заново;
- лучший успешный результат сохраняется.

## Пример расчета

Если в тесте 4 вопроса, а пользователь ответил правильно на 3, результат считается так:

```text
3 / 4 * 100 = 75%
```

Если проходной балл равен 70%, тест считается пройденным.

## Сертификат

После прохождения всех уроков открывается страница поздравления.

На ней пользователь может скачать PDF-сертификат.

В текущем MVP сертификат простой:

- белый лист A4;
- имя пользователя в верхнем левом углу;
- файл скачивается повторно без ограничений.

Такой подход подходит для демонстрации полного сценария: от входа в систему до завершения курса.
