# Предложения по улучшению и развитию системы

## Backend

- Добавить автоматические тесты для авторизации, пользователей, уроков и прогресса.
- Сделать отдельную страницу редактирования контента - можно будет обновлять курс без перезапуска backend.
- Добавить логирование запуска, импорта контента и ошибок API.
- Добавить пагинацию для списка пользователей - админка не будет деградировать при росте числа студентов.
- Добавить фильтры по роли и статусу пользователей - администратору будет проще управлять списком.
- Запретить архивирование текущего администратора самим собой (Смешной баг).
- Сделать более подробный PDF-сертификат - добавить название курса, дату выдачи и номер сертификата.
- Добавить таблицу выданных сертификатов - можно будет хранить историю выдачи и статистику.

## Frontend

- Добавить обработку истекшей сессии - пользователь будет автоматически возвращаться на страницу входа.
- Улучшить отображение ошибок API - вместо технических сообщений показывать понятный текст.
- Улучшить страницу завершения курса - сделать скачивание сертификата более заметным.
- Улучшить отображение markdown-уроков на мобильных экранах - читать материалы будет удобнее.
- Добавить сохранение последнего открытого урока - студенту проще продолжать обучение.
- Добавить прогресс по каждому уроку прямо в списке - студент быстрее понимает свое состояние.

## Контент и обучение

- Добавить описания уроков и примерную длительность - студенту проще планировать обучение.
- Добавить разные типы вопросов - например несколько правильных ответов или открытый ответ.

## Администрирование

- Добавить учебные группы - удобно разделять студентов по потокам или классам.
- Добавить назначение курса группе - можно будет управлять доступом массово.
- Добавить экспорт пользователей и прогресса в CSV/XLSX - полезно для отчетности.
- Добавить просмотр прогресса конкретного студента - администратор увидит, где студент остановился.
- Добавить кабинет общей статистики прогресса студентов.
- Добавить поиск пользователей по имени и логину - быстрее работа с большой группой.
